# Custom_Networks_Template
This is a repo to base fuondation of another custom networks
--------

### how to run:
- `setup env`:
```bash
conda env create -f environment.yaml
```
- activate env:
```bash
conda activate custom_networks_template
```

```
cd Custom_Network_Template
source init.sh
```
